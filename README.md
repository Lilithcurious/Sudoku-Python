# Sudoku em Python

Um jogo de Sudoku completo com:
- Geração automática de puzzles
- Validação de jogadas
- Interface no terminal
- Solução automática com backtracking

Formato da jogada:
linha coluna número → Ex: 1 2 5
Comandos:
sol - Mostra a solução / 
sair - Encerra o jogo
